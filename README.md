## 🌟 Features

- **Dark Hacker Theme** - Ultra-modern black background with neon green/blue glows
- **Glassmorphism Cards** - Beautiful frosted glass effects
- **Smooth Animations** - Motion React powered animations
- **Typing Animation** - Dynamic hero section text
- **Animated Progress Bars** - Interactive skill showcases
- **Security Tools** - Password Generator & Text Encryption/Decryption
- **Mobile Responsive** - Fully responsive design
- **Dark Mode Toggle** - Light/Dark theme switcher
- **Live Clock** - Real-time display
- **Terminal Animation** - Cool hacker-style terminal

## 🛠️ Tech Stack

- **React** - UI Framework
- **TypeScript** - Type Safety
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Motion React** - Animations
- **Lucide React** - Icons
